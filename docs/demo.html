Warum das zählt

  

    Der Generator (die KI) darf Code schreiben, aber nicht selbst „grün" vergeben — das macht ein getrenntes Orakel (Generator ⊥ Orakel). Das alte Marker-Orakel lässt sich täuschen: ein leeres Skelett trägt den Marker und gilt als fertig. Das harte Gate verlangt einen echten grünen Lauf — Fake-Grün wird blockiert. Aber: ein trivialer Test ist grün und kommt durch — „grün heißt: die geschriebenen Tests passen, nicht: korrekt" (Rice/Dijkstra). Deshalb bleibt das Setzen einer tragenden, scharfen Spec Menschensache. Genau diese Ehrlichkeit ist der Punkt — eine Schraube weiter, nicht „gelöst".

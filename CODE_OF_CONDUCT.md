# Code of Conduct

Dieses Projekt folgt dem [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Kurzfassung: Sei respektvoll, konstruktiv und geduldig. Keine Belästigung,
keine Herabwürdigung, keine Trollerei — in Issues, PRs, Discussions und allen
anderen Projekträumen.

Verstöße bitte vertraulich an den Maintainer melden: **@Koschnag**
(via GitHub-Profil). Meldungen werden zeitnah geprüft; Konsequenzen reichen
von Verwarnung bis Ausschluss aus dem Projekt.

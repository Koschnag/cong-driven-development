CDD Cockpit

So funktioniert das Cockpit

  1. 🧰 Toolbox: Knotenart anklicken → Formular ausfüllen → Speichern. Fertig.
  2. 📋 SPOT-Explorer: Alle Knoten, filterbar. Klick öffnet den Inspektor.
  3. 🔍 Inspektor: Lesen & springen — Verlinkungen anklicken, ←/→ wie im Browser. ✏️ Bearbeiten öffnet das Formular.
  4. 🧊 Cube-Bar: Filter-Chips (Art, Status) + Fokus = nur die Nachbarschaft des gewählten Knotens in Graph/UML.
  5. Sichten: Dashboard (Überblick) · Graph & UML (Diagramme, klickbar) · Doku (generiert, für KI kopierbar) · 🤖 Agent (Änderungen in Prosa beschreiben).

Farben: umgesetzt · offen · abweichend · ohne Modell

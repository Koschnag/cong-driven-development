Bereit

So funktioniert das Cockpit

  1. Phasen-Leiste oben: der Lebenszyklus jeder Entität (Pending → Diverged → Orphaned → Aligned). Klick filtert die Navigation.
  2. Navigation links: nach Knotenart gruppiert. Klick auf einen Knoten öffnet ihn im Hauptfenster.
  3. Hauptfenster (geteilt): links die Nachbarschaft als Diagramm, rechts die Details — Kriterien, Beziehungen (klickbar), Konvergenz, Aktionen.
  4. Unten: Fehlerliste (Klick springt zum Knoten) · Agent (Prosa → Modelländerung) · Dashboard.

Farben: umgesetzt · offen · abweichend · ohne Modell
